## What I Learned

Today I realized that the terminal is simply an interface for interacting with the shell. The shell is responsible for interpreting commands and executing programs.

One of the biggest concepts I learned was `PATH`. When I type a command like `git` or `python3`, the shell searches each directory listed in `PATH` until it finds the corresponding executable. If the executable exists but its directory is not listed in `PATH`, the shell cannot find or execute it.

I also learned that Linux is technically a kernel, while distributions such as Ubuntu and Debian package the Linux kernel together with additional software to create a complete operating system.
