# HTTP

## What is the Internet?
The internet is a global network of computers , all connected to each other with the ability to communicate with each other. The internet is the underlying network to the web. The World Wide Web is one service that runs on the Internet.

## What is an IP address?
An IP address is a unique number that is assigned to a device on the internet so data so data knows where to be delivered. 

## What is DNS?
DNS stands for Domain Name Sytem. It takes domain names and translates them into IP addresses. For example, lets say you type google.com. Your computer doesn't know where that is so it asks a DNS server what the IP address for google.com is. The DNS server reponds with an IP address so your browser can connect to it. 

## What is HTTP?
HTTP stands for Hyper Text Transfer Protocol. It is the set of rules for how a client and a server communicate on the web. When your browser wants a webpage, it will send a HTTP request. The server then recieves it, processes it, and sends back an HTTP response. 

## What is HTTPS?
HTTPS stands for Hyper Text Transfer Protocol Secure. It is basically HTTP with the addition of encryption. Data that is sent back and forth is encrypted by TLS. 

## What is a client?
A client is anything that is sending a request and awaits a response. Most commonly a browser requesting a webpage. 

## What is a server?
A server is a program running on a computer that listens for requests and sends responses. It's just a program running on a machine that sends reponses based on client requests. 

## What is an API?
An API is an Application Programming Interface. It is a defined way for two programs to communicate. On the web it is usually a server that accepts HTTP requests and returns data such as JSON.

## What is JSON?
JSON stands for JavaScript Object Notation. It is a text format for presenting structured data. It is the most common format that APIs use to send data. 

## Things That Surprised Me
- The Internet and the World Wide Web are not the same thing.
- A server is a program, not necessarily a dedicated computer.
- My browser never connects directly to "google.com". DNS first translates it into an IP address.
